# 🎯 Web Quiz Game

Um jogo de quiz interativo desenvolvido utilizando **HTML, CSS e JavaScript puro**, com foco em prática de lógica e organização de código no front-end.

---

## 🚀 Funcionalidades

* ✅ 4 categorias de perguntas
* ✅ 3 níveis de dificuldade:
  * Fácil
  * Médio
  * Difícil
* ✅ Sistema de configuração de som e modo escuro
* ✅ Interface interativa
* ✅ Pontuação
* ✅ Feedback de respostas (correta/incorreta)

---

## 🧠 Categorias disponíveis

* 📚 Curiosidades
* 🌍 Música
* 🔬 Animações
* 🎮 Jogos

---

## ⚙️ Configurações do jogo

Antes de iniciar, o usuário pode escolher:

* Categoria das perguntas
* Nível de dificuldade
* Modo de jogo

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar JavaScript puro (sem frameworks)
* Criar lógica de jogo
* Organizar código front-end
* Desenvolver uma aplicação interativa simples
* Mínimo de vibe-coding
